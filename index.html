O Projeto

Monitoramento Autônomo de Desmatamento

Nosso projeto utiliza um Raspberry Pi 2W Pico e diversos sensores para criar um sistema autônomo de monitoramento florestal capaz de detectar atividades de desmatamento em tempo real.

Através da análise de níveis de CO₂ e reconhecimento de sons de motosserra, nosso dispositivo pode alertar sobre atividades suspeitas e contribuir para a preservação das florestas.

Principais características:

  • Sensores de CO₂ para monitorar alterações ambientais
  • Microfones para detecção de sons de motosserra
  • Sistema alimentado por ...
  • Comunicação sem fio para envio de alertas
  • Interface web para monitoramento remoto

Detecção de CO₂

Nossos sensores são capazes de detectar mudanças nos níveis de CO₂, que podem indicar queimadas ou outras atividades de desmatamento. O aumento repentino de CO₂ é um forte indicador de degradação florestal.

Reconhecimento de Sons

Utilizamos tecnologia de reconhecimento de áudio para identificar sons específicos como motosserras e veículos pesados, que são comuns em operações de desmatamento ilegal.

Monitoramento Ambiental

Além da detecção de atividades ilegais, nosso sistema também monitora condições ambientais como temperatura, umidade e qualidade do ar, contribuindo para estudos ambientais de longo prazo.

Sobre o Projeto

Nosso sistema de monitoramento representa um avanço significativo na luta contra o desmatamento ilegal, combinando tecnologia acessível com monitoramento ambiental eficiente.

O desmatamento é uma das principais causas das mudanças climáticas e da perda de biodiversidade. Pequenas áreas de desmatamento ilegal muitas vezes passam despercebidas por satélites, mas nosso sistema de sensores pode detectar essas atividades localmente e em tempo real.

Como nosso projeto ajuda no combate ao desmatamento?
  • Detecção precoce: Alertas em tempo real sobre atividades suspeitas, permitindo ação rápida.
  • Monitoramento constante: Funcionamento autônomo, necessidade moderada de intervenção humana.
  • Baixo custo: Utilizamos componentes acessíveis como o Raspberry Pi 2W Pico, tornando o sistema replicável.
  • Coleta de dados: Geração de dados valiosos para pesquisas ambientais e planejamento de conservação.
  • Complemento a outras tecnologias: Trabalha em conjunto com monitoramento por IA para uma cobertura mais completa.

A Construção

Nosso projeto utiliza componentes de hardware e software de código aberto para criar um sistema de monitoramento eficiente e replicável. Abaixo você pode visualizar o modelo 3D da case do nosso protótipo.

Componentes Utilizados

  • Raspberry Pi 2W Pico Microcontrolador central.
  • Sensor MQ-2 Detecção de CO₂ e gases.
  • Sensor de Chama Detecção de Chamas.
  • Sensor DHT22 Detecção da temperatura do ambiente.
  • Microfone MAX9814 Detecção de som ambiente.
  • Painel Solar 5V Alimentação energética.
  • Módulo SIM800L Comunicação sem fio com uso de SMS.
